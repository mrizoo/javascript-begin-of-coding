Type conversion

Type conversion - Javascript dasturlash tilitda saqlanayotgan malumot turini bir ko'rinishi ikkinchi ko'rinishiga o'tkazish

Number

Number - primitiv o'rab turuvchi obyektdan foydalanib, berilgan ma'lumot turini son ko'rinishiga o'tkazish mavjud. Agar berilgan ma'lumot son ko'rinishiga o'tkazish imkni bo'lmasa, NaN maxsus qiymatini yuzaga keladi

const yosh = '23'; // string

console.log(yosh); // string

console.log( Number(yosh)); // Number

console.log(yosh+1); // 231 (concantenation)

console.log(Number(yosh)+1); // 24

const ism = 'Muhammadrizo';

console.log(Number(ism)); // Nan

String

String - primitiv o'rab turuvchi obyektidan foydalanib, berilgan ma'lumot turini string ko'rinishiga o'tkazishb imkoniyati mavjud.

const yil = '2024'; // Number

console.log(yil); // Number

console.log(String(yil)); // String

Type coercion

type coercion - Javascript dasturlash tilida operator turli xil qiymatlar ustida ish olib borayotganda, ularda birinign avtomatik ravishda boshqa turga o'tishi. Boshqa qilib aytganda ma'lumot turini avtomatik ravishda bir ko'rinishdan ikkinchi ko'rinishga o'tish

Auto string conversion

console.log('Men' + 2005 + 'chi yilda tug'ilganman''); // auton string son

Auto number conversion

console.log('30' - '10' - 5 ); auto string conversion

Truthy va Falsy

Javascript dasturlash tilida berilgan qiymatni Boolean ko'rinshiga o'tkazilganda ular Truthy bu true ya'ni rist qiymatga ega bo'gan ma'lumotlar, Falsy esa false qiymatiga ega bo'laganlarga aytiladi

Falsy ko'rinishida keladigan qiymatlar

  • false
  • 0
  • ''
  • undefined
  • null
  • NaN
  • On

//Falsy qiymatlar

console.log(Boolean(false));

console.log(Boolean(0));

console.log(Boolean(''));

console.log(Boolean(undefined));

console.log(Boolean(null));

console.log(Boolean(Nan));

console.log(Boolean(On));

//Truthy qiymatlar

console.log(Boolean(Muhammadrizo));

console.log(Boolean(18));

console.log(Boolean([]));

console.log(Boolean({}));

console.log(Boolean('0'));

console.log(Boolean(false));
